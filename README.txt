MEncryptor is a free encryption tool with many protocols and a frequency analyzer to break any monoalphabetic system.
The project uses Qt 6 opensource.

"src" folder contains sources, "lang" folder contains language resources and "release" folder containsall files needed to use the application (images, data...)
